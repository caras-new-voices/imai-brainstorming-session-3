# InfluencerMarketing.ai — Strategic Revenue Funnel & Growth Workshop

A single-page, tabbed website presenting the strategic synthesis of the
InfluencerMarketing.ai revenue funnel and growth workshop.

## What's inside

The synthesis is organized into eleven navigable sections rather than one long
scroll:

1. **Overview** — executive summary, agreement points, meeting context & objectives
2. **Discussion Themes** — the twelve themes from the session
3. **Strategic Insights** — seven cross-cutting insights
4. **Best Opportunities** — seven prioritized opportunities
5. **Alignment & Open Questions** — areas of agreement and unresolved issues
6. **Risks & Constraints** — explicit and inferred risks
7. **Market & Product** — customer, product, and positioning insights
8. **Important Quotes** — key quotes and why they matter
9. **Action Items** — explicit and recommended actions with owners
10. **Recommended Steps** — a phased roadmap
11. **Final Takeaways** — conclusions and information worth preserving

## Tech

A single self-contained `index.html` (embedded CSS + vanilla JS). No build step.
Responsive for mobile and desktop, with a sidebar navigation on desktop that
collapses to a drawer on mobile.

## Local preview

Open `index.html` directly in a browser, or serve the folder:

```bash
npx serve .
```

## Deployment

Deployed as a static site on Vercel — the root `index.html` is served directly,
no configuration required.
